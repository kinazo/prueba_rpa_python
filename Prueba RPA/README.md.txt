# My RPA Project

Este proyecto es una demostración de un flujo de trabajo de automatización utilizando Python. El proyecto incluye:

- **PowerShell**: Ejecución de scripts de PowerShell desde Python.
- **API**: Creación y consumo de una API con Flask.
- **Web Scraping**: Extracción de datos de un sitio web utilizando Selenium.
- **Automatización de escritorio**: Control de aplicaciones de escritorio con Pywinauto.

## Instalación

1. Clonar el repositorio.
2. Crear un entorno virtual y activar:
    ```bash
    python -m venv venv
    source venv/bin/activate # En Windows usa `venv\Scripts\activate`
    ```
3. Instalar las dependencias:
    ```bash
    pip install -r requirements.txt
    ```

## Uso

1. Ejecuta el script principal:
    ```bash
    python main.py
    ```
